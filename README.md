# Urban Rides - Backend

Urban Rides is a car rental web application where users can browse available cars, apply filters, book cars, and manage their bookings & can earn by linsting their cars for rent.

This repository contains the React frontend of the application.

## Project Goal

The goal of Urban Rides is to provide users with a smooth and modern car rental experience through an interactive and responsive interface.

The frontend focuses on clean UI design, seamless navigation, real-time interactions, filtering functionality, booking flow management, and an overall user-friendly experience for both renters and vehicle owners.
